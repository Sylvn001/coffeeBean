# CoffeeBean

> A basic authentication project using Ruby, without using database persistence, just using in-memory structures.

### 💻 Ruby version - 3.0.3

### ⚒️ Tecnologies
```
  *Ruby
  *HTML
  *CSS
  *Javascript
```

### 🚀 How start
```
  bundle install
  rails s -p 3000
  open in bowser localhost:3000/
```
